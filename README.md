# PyRadioLib

Домашний проект по портированию библиотеки [RadioLib](https://github.com/jgromes/RadioLib) в python.

### Для чего это, предыстория

Я хочу собрать универсальный ретранслятор для сети [Meshtastic](https://meshtastic.org/). Для его нормальной работы
необходимо иметь возможность работать сразу с двумя радио модулями E22-900M30S (Две разные антенны). Я хорошо знаю
python и понимаю его возможности. В качестве платформы я хочу использовать Raspberry Pi (любая версия или совместимые
платы).

Проанализировав исходники Meshtastic я выделил основных этапа для повторения
функционала и создания прошивки:

1. Аппаратная поддержка - в библиотеки уже есть реализация
   под [Raspberry Pi](https://github.com/jgromes/RadioLib/tree/master/src/hal/RPi).
2. Управление через RadioLib - Текущий этап портирования.
3. Реализация алгоритмов шифрования Crypto lib (как я понял в основе libssl) - В планах разобраться, что нужно и
   как это запустить в python.
4. Protobuf сериализация - *.proto доступны в исходниках, python умеет работать с такими файлами и создавать интерфейсы.
5. Роутер - Основной алгоритм направления пакетов
6. Модули - Конечные обработчики пакетов
